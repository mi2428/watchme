# ``WatchmeWiFi``

WatchMe Wi-Fi observability support for the `watchme wifi` command.

## Overview

`WatchmeWiFi` owns the Wi-Fi subcommand, CoreWLAN and SystemConfiguration event monitors, Wi-Fi snapshot collection, active HTTP probes, and passive BPF packet timing.
For the full instrumentation map, see <doc:WatchMeWiFiObservability>.
